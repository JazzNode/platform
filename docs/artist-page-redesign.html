實作優先順序建議

#項目理由開發量
1Hero 重新設計
Cover + Headline + Stats + Status		第一印象決定一切。「10 秒法則」—— 訪客瞬間理解這個藝人是誰低
2Media 區上移把 Spotify/YouTube 從中段移到 About 之後。音樂是核心極低
3Upcoming Shows 改版
日曆式列表		比卡片網格更清楚、更適合掃描，票務連結更突出低
4Quick Facts + Similar Artists增加頁面資訊密度和站內導流低
5Profile 完成度（owner only）驅動藝人填寫更完整的資料，提高整體頁面品質低
6推薦語系統社交證明是 EPK 核心。需要新表 + 審核，但價值極高中
7EPK 下載按鈕API 已有，只需加一個 icon button 到 Hero極低
