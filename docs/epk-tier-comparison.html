Tier 3 · 品牌版

深色奢華、Hero 封面、媒體素材、QR Code、白標

設計筆記

🏷️ Tier 1 — Claimed (免費)

  • 自動從 profile 資料產生，零設定
  • 白底灰字，乾淨但樸素，讓人想升級
  • 進階區塊顯示「🔒 Premium」模糊遮罩
  • Footer 固定顯示 Generated by JazzNode
  • PDF 下載只在 Dashboard 可用
  • 無公開分享連結

⭐ Tier 2 — Premium (付費)

  • 金色主題提升質感、Premium 標章
  • 新增數據摘要列：瀏覽數、粉絲、演出、評價
  • 解鎖器材展示與接案資訊
  • 可加入精選演出（手動排序 Top 6）
  • 提供公開分享連結（Web EPK）
  • 追蹤 EPK 瀏覽次數
  • Footer 僅小字 JazzNode

👑 Tier 3 — Elite (品牌)

  • 深色奢華版面或自訂品牌配色
  • Hero 封面區含藝人頭像 / Logo
  • 新增媒體引言 / 推薦語區塊
  • 新增媒體素材區（專輯封面、劇照）
  • Booking QR Code 直接掃碼聯繫
  • Footer 可完全白標（移除 JazzNode）
  • 自訂 Footer 文字（© Artist Name）
  • 未來可擴展：自訂字型、多頁 PDF

💡 升級誘因設計（Upsell Triggers）

Tier 1 → 2：基本版 EPK 中，器材與接案區塊以「🔒 解鎖 Premium」模糊遮罩呈現，讓藝人看到內容被隱藏而想升級。數據摘要列也只在 Premium 才顯示。
Tier 2 → 3：Premium 版 EPK 的 Footer 仍帶 JazzNode 品牌，藝人頭像和媒體素材區無法使用。Elite 版提供完全白標體驗 + QR Code + 媒體引言，適合專業藝人作為正式經紀資料使用。
